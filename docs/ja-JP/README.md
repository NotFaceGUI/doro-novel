## 使用ガイド

### 📦 ダウンロードとインストール

1. **アプリケーションのダウンロード**
   - [リリースページ](https://github.com/NotFaceGUI/doro-novel/releases/latest)にアクセス
   - お使いのオペレーティングシステムに対応するバージョンをダウンロード：
     - Windows: `doro-novel_x.x.x_x64-setup.exe`
     - macOS: `doro-novel_x.x.x_x64.dmg`
     - Linux: `doro-novel_x.x.x_amd64.AppImage`

2. **アプリケーションのインストール**
   - Windows: `.exe`ファイルをダブルクリックし、インストールウィザードに従ってインストール
   - macOS: `.dmg`ファイルを開き、アプリをApplicationsフォルダにドラッグ
   - Linux: `.AppImage`ファイルに実行権限を追加して直接実行

### 🎮 使い始める

1. **アプリケーションの起動**
   - 初回起動時にウェルカム画面と使用ガイドが表示されます

2. **ファイル関連付け**
   - アプリケーションは`.doro`、`.Doro`、`.DORO`形式のプロジェクトファイルをサポート
   - インストール後、これらのファイルをダブルクリックしてプロジェクトを開くことができます

3. **リソース管理**
   
   アプリケーションのリソースファイルは`resources`フォルダ内にあり、以下のディレクトリが含まれます：
   
   - **📁 audio** - オーディオリソースフォルダ
     - 背景音楽、効果音などのオーディオファイルを保存
   
   - **📁 character** - キャラクターリソースフォルダ
     - さまざまなキャラクターのSpineファイルを含む

     ---
     
     ### 🚀 キャラクターリソース自動更新スクリプト
     
     最新のキャラクターリソースを簡単に取得するため、クロスプラットフォーム対応の自動更新スクリプトを提供しています：
     
     #### Windows ユーザー
     ```powershell
     # アプリケーションインストールディレクトリで実行
     .\update_l2d.ps1
     ```
     
     #### Mac ユーザー
     ```bash
     # アプリケーションインストールディレクトリで実行
     ./update_l2d_mac.sh
     ```
     
     #### Linux ユーザー
     ```bash
     # アプリケーションインストールディレクトリで実行
     ./update_l2d_linux.sh
     ```
     
     #### スクリプト機能説明
     
     これらのスクリプトは以下の操作を自動的に実行します：
     
     1. **Git環境チェック**
        - Windows: MinGit ポータブル版を自動ダウンロード・解凍（約45MB）
        - Mac/Linux: システムGitインストールをチェックし、インストールガイダンスを提供
     
     2. **リソースリポジトリのクローン/更新**
        - [Nikke-db/Nikke-db.github.io](https://github.com/Nikke-db/Nikke-db.github.io) から最新のL2Dリソースをクローン
        - 既に存在する場合は `git pull` を実行して最新版に更新
     
     3. **スマートファイル同期**
        - ソースリポジトリの `l2d` ディレクトリとローカルの `resources/character` ディレクトリを比較
        - 不足しているファイルのみをコピーし、重複ダウンロードを回避
        - ディレクトリ構造を完全に維持
     
     4. **ダウンロード内容**
        - 各キャラクターのSpineアニメーションファイル（.skel, .atlas, .png）
        - キャラクター立ち絵と表情リソース
        - 推定総サイズ：3GB+（キャラクター数に依存）
     
     #### 🙏 謝辞
     
     オープンソースリソースリポジトリ [Nikke-db.github.io](https://github.com/Nikke-db/Nikke-db.github.io) を維持している [Nikke-db](https://github.com/Nikke-db) チームに特別な感謝を表します。コミュニティに完全なキャラクターリソースデータを提供してくださっています。このリポジトリにはゲーム内すべてのキャラクターの高品質Spineアニメーションリソースが含まれており、本プロジェクトがユーザーに豊富なキャラクター表示機能を提供することを可能にしています。
     
     彼らの無私の貢献と継続的なメンテナンスがなければ、本プロジェクトのキャラクターシステムはこれほど完全な機能を実現することはできませんでした。

     ---
   
   - **📁 image** - 画像リソースフォルダ
     - 背景画像や立ち絵などを保存
     - `Background` サブフォルダには背景画像を保存
     - `x2` サブフォルダには高解像度リソースを保存
   
   - **📁 locales** - 多言語フォルダ
     - 中国語 `zh-CN`、英語 `en-US`、日本語 `ja-JP`、韓国語 `ko-KR` をサポート
   
   - **📁 package** - パッケージリソースフォルダ
     - シェーダーなどの特殊リソースを含む
     - あるいは詳細な JSON ファイル

   - **📁 video** - ビデオリソースフォルダ
     - アニメーションビデオファイルを保存
   
   - **📄 spine-character.json** - Spineキャラクター設定ファイル
     - キャラクターアニメーションとスケルタル情報を定義

### 💡 使用のコツ

- リソースファイルのドラッグ＆ドロップインポートをサポート
- Spineアニメーション効果のリアルタイムプレビューが可能
- 多言語インターフェース切り替えをサポート
- ファイル整理に便利な内蔵リソースマネージャー
